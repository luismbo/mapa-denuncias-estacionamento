# mapa-denuncias-estacionamento

Este permite visualizar os dados públicos da aplicação [Denúncia de Estacionamento](https://denuncia-estacionamento.app) apresentando-os num mapa para identificação de pontos críticos de estacionamento ilegal.

Objetivo

- Facilitar a análise espacial de denúncias para identificar locais com maior incidência.
- Ajudar investigados, coletivos cidadãos e autoridades a priorizar fiscalizações.

Fonte dos dados

Os dados são obtidos a partir da API pública e gratuita do projeto [Denúncia de Estacionamento](https://denuncia-estacionamento.app), que é uma aplicação móvel para submissão de queixas de estacionamento ilegal junto da autoridade policial, ao abrigo do n.º 5 do art.º 170.º do Código da Estrada (Portugal).

- Documentação da API: https://denuncia-estacionamento.app/api.html
- Repositório do projeto fonte: https://github.com/jfoclpf/form-for-parking-violation
- Política de Privacidade: https://denuncia-estacionamento.app/privacy-policy.html
